# employee-detail-crud-system

## Description
A simple Python-based project to store, add, update, and manage employee records using a CSV file.

## Features
- Add new employees
- View employee list
- Update employee details
- Delete employee records
- Data stored in CSV format

## Technologies Used
- Python
- CSV file handling

## How to Run
1. Download the repository.
2. Open `employees.py` in any Python IDE or VS Code.
3. Run the program using:
   ```bash
   python employees.py

# project file
   employees.py - main code
   employees.csv - data storage file
